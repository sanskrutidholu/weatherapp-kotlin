Weather app using in kotlin using Retrofit Library to fetch data from APIs

Objects used to show weather of a particular location are :

-> Country Name

-> City Name

-> Searched Date

-> Location (lat, long)

-> Sunset & Sunrise time

-> Wind Speed & Degree

-> Temperature

-> Humidity
